# BleRDF

## 概要
Foxハンティング(or ARDF)をBluetooth Low Energy (BLE)でやる試み

ARDFにはいくつかの課題があります。
- ビーコン設置に免許が必要
- アマチュア無線の帯域を利用するため、商用利用が不可
- 送信機・受信機のコストが高い
- 手軽でなくハードルの高さを感じる

そこで、ISMバンドかつ一般的なBLEを利用して同様の遊びを実現しようとしています。

### 比較
 
| | ARDF | BleRDF |
| ---- | ---- | ---- |
| 送信受信機の費用 | 数万円 | 数千円 |
| 利用する周波数 | 3.5MHz / 144MHz | 2.4GHz |
| 空中線出力 | 3.5MHz(3-5W) / 144MHz(0.25-1.5W)程度 | 0.06-8mW |
| 活動範囲 | 数キロメートル | 数百メートル |
| 受信機での探知方法 | 音 | 数値 |
| 受信アンテナの改造 | 可能 | 不可 |
| 競技規模 | 世界大会あり | ただの遊び |
